# soco-cli Aliases Configuration
# Copy this file to your config/.soco-cli/ directory

## Basic Aliases
# p -> play
# s -> pause
# v -> volume
# n -> next
# b -> previous
# l -> list
# i -> info

## Sequence Examples
# Use ':' to chain multiple commands
# start -> play:volume 30
# stop -> pause:volume 0

## Usage
# In interactive mode, type alias directly:
# > p              # plays music
# > start          # plays and sets volume to 30
